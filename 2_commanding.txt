2. change (c)

	The change command works like delete,
	but puts you straight into Insert mode.

	c = delete + insert

	cw   - change word
	ce   - change to end of word
	cc   - change entire line
	c$   - change to end of line - Alternative is C

3. yank and paste (y & p)

	Yanking copies text.
	Pasting puts it after the cursor.

	y + motion   - yank text
	yy           - yank line
	p            - paste after cursor
	P            - paste before cursor

4. indenting (< & >)

	Indenting is very useful for code and structured text.

	>  - indent right
	<  - indent left
  >> - indent line
	<< - unindent line

	These also work with motions.
